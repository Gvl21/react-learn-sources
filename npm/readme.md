# npm (Node Package Manager)
- npmjs.com 에서 라이브러리 검색 가능

## 패키지 시작
- npm init

## 패키지 실행
- npm run start

## 외부 라이브러리 설치하기
- npm i (라이브러리명)

## package.json
- 패키지 기본 정보
- 모듈을 ESM 방식으로 사용할 때
- "type" : "module" 추가

## ESM 방식 모듈
- import {... .. } from "파일명"
- export 변수, export 함수
- export default 함수
- import 함수(이름 변경 가능) from "파일명"
